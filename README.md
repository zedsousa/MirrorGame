# MirrorGame

Controles

Movimento
WASD

Tiro
Botão esquerdo do mouse

O jogador deve atirar nos alvos para desbloquear o caminho até chegar ao final do jogo.

No meio do caminho há moedas que o jogador poderá coletar, as moedas continuam com o jogador mesmo se ele sair do jogo e voltar depois.

O objetivo da moeda era de ser utilizada para comprar munição nos pontos de venda espalhados pelo mapa, mas isso não foi implementado.

Se o jogador não tiver completado o jogo e sair, os caminhos que ele abriu continuarão liberados, mas ainda assim é necessário atirar nos alvos para que ele atirou no passado.

O jogo possibilita que outros jogadores entrem na aventura para cooperar.

Para colocar outro jogador é necessárioir em ParrelSync > Clones Manager > Open in New Editor
